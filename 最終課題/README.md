# 最終課題

このプロジェクトでは、航空会社のロストバゲージに関するデータをスクレイピングし、
データ分析を行って仮説を検証します。

## ディレクトリ構成

- data/
  - raw_data.csv
- figures/
- notebooks/
  - analysis.ipynb
- scripts/
  - scrape_data.py
  - data_analysis.py
- README.md
- requirements.txt
